–°–æ–≤–µ—Ç—É–µ—Ç—Å—è –∏–ª–∏ –∏–∑–±–µ–≥–∞—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤ ¬´–≥—Ä—É–±–æ–π —Å–∏–ª—ã¬ª –∏–ª–∏ –Ω–∞—Ö–æ–¥–∏—Ç—å —Ä–µ—à–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–µ —É—Å–∫–æ—Ä—è—é—Ç –∏—Ö —Ä–∞–±–æ—Ç—É.

### 3.4 –†–µ–∫—É—Ä—Å–∏–≤–Ω—ã–µ –∞–ª–≥–æ—Ä–∏—Ç–º—ã

 –û–±—ã—á–Ω–∞—è —Ä–µ–∫—É—Ä—Å–∏—è –∏–º–µ–µ—Ç —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω—É—é –∞–ª–≥–æ—Ä–∏—Ç–º–∏—á–µ—Å–∫—É—é —Å–ª–æ–∂–Ω–æ—Å—Ç—å.

- —Å—É–º–º–∞ –Ω–æ–º–µ—Ä–æ–≤ —Å—Ç–µ—Ä–∂–Ω–µ–π `1 + 2 + 3` –≤—Å–µ–≥–¥–∞ = **6**.
- —Å–ª–æ–∂–Ω–æ—Å—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º–∞ **—ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–∞—è**.

###### –§–æ—Ä–º—É–ª–∞ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –∫–æ–ª-–≤–∞ –ø–µ—Ä–µ–º–µ—â–µ–Ω–∏–π –¥–∏—Å–∫–æ–≤
–í–≤–µ–¥–µ–º —Ñ—É–Ω–∫—Ü–∏—é: `T(n) = 2*T(n - 1) + 1`.
–ù–∞—á–∏–Ω–∞—è —Å `T(1) = 1` - —ç—Ç–æ —Ä–µ–∫—É—Ä—Ä–µ–Ω—Ç–Ω–æ–µ —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏–µ ( #—Ä–∞–∑–æ–±—Ä–∞—Ç—å) –∑–∞–¥–∞–µ—Ç—Å—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å—é: `1, 3, 7, 15, 31, 63 ..`

–î–ª—è –ø—Ä–∏–≤–µ–¥–µ–Ω–∏—è —Ñ–æ—Ä–º—É–ª—ã –∫ –¥—Ä—É–≥–æ–º—É –≤–∏–¥—É –ø—Ä–∏–±–∞–≤–∏–º **1** –∫ `T(n)` :
`T(n) + 1 = 2*(T(n - 1) + 1)`

–î–æ–±–∞–≤–∏–º –Ω–æ–≤–æ–µ –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏–µ: ¬†`U(n) = T(n) + 1`
–¢–æ–≥–¥–∞: `U(n) = 2*U(n - 1)`
–≠—Ç–∞ —Ñ-–∏—è, –Ω–∞—á–∏–Ω–∞—è —Å¬†`U(1) = 2` –¥–∞–µ—Ç –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å: ¬†`2, 4, 8, 16, 32, 64 ..`
–ê —ç—Ç–∞ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –Ω–∏ —á—Ç–æ –∏–Ω–æ–µ, –∫–∞–∫ `2^n`.

–ü–æ–ª—É—á–∞–µ–º:

- `T(n) = U(n) ‚àí 1 =` **2^n ‚àí 1** (—ç–∫—Å–ø–æ–Ω–µ–Ω—Ç–∞).

###### –§–æ—Ä–º—É–ª–∞ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –∫–æ–ª-–≤–∞ –ø–µ—Ä–µ–º–µ—â–µ–Ω–∏–π –¥–∏—Å–∫–æ–≤

> (2*`(4 —Å—Ç–µ—Ä–∂–Ω—è)` + 1*`(3 —Å—Ç–µ—Ä–∂–Ω—è)`)

–ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –∏–∑ –Ω–∞–∏–º–µ–Ω—å—à–µ–≥–æ –∫–æ–ª-–≤–∞ –ø–µ—Ä–µ–º–µ—â–µ–Ω–∏–π –¥–ª—è `1 ‚â§ n ‚â§ 10` –¥–∏—Å–∫–æ–≤ –≤—ã–≥–ª—è–¥–∏—Ç —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º: `1, 3, 5, 9, 13, 17, 25, 33, 41, 49`.

–§–æ—Ä–º—É–ª—ã –¥–ª—è —á–∏—Å–µ–ª –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ (–ø—Ä–∏ –∑–∞–π–¥–µ—Å—Ç–≤–æ–≤–∞–Ω–∏–∏ 4-—Ö —Å—Ç–µ—Ä–∂–Ω–µ–π):
- \[`T(2)`, `T(4)`\] –≤—ã—Ä–∞–∂–∞–µ—Ç—Å—è —Ñ–æ—Ä–º—É–ª–æ–π: **`2^n-1 + 1`**.
- \[`T(5)`, `T(7)`\] : **`2^n-3 + 9`**.

---
> –†–∞–∑–±–µ—Ä–µ–º –ø–æ–¥—Ä–æ–±–Ω–µ–µ –∞–ª–≥–æ—Ä–∏—Ç–º –¥–ª—è `T(n) = T(4)`:

1. –ú—ã –ø–µ—Ä–µ–º–µ—â–∞–µ–º –≤—Å–µ –¥–∏—Å–∫–∏ –∫—Ä–æ–º–µ 2-—Ö (**`k`** = `n - 2`) –Ω–∞ —Å–≤–æ–±–æ–¥–Ω—ã—Ö —Å—Ç–µ—Ä–∂–µ–Ω—å. –ü—Ä–∏  —ç—Ç–æ–º —É –Ω–∞—Å –∑–∞–¥–µ–π—Å—Ç–≤–æ–≤–∞–Ω—ã –≤—Å–µ **4 —Å—Ç–µ—Ä–∂–Ω—è**.
   –ú—ã –ø–µ—Ä–µ–º–µ—â–∞–µ–º `T(k) = n - 2 = 4 - 2 =` **2** –¥–∏—Å–∫–∞.
   T.–µ. `T(k) = T(2)` **`->`** –≤–æ—Å–ø–æ–ª—å–∑—É–µ–º—Å—è —Ñ–æ—Ä–º—É–ª–æ–π **`2^k-1 + 1`**.
2. –î–∞–ª–µ–µ –Ω–∞–º –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø–µ—Ä–µ–Ω–µ—Å—Ç–∏ –æ—Å—Ç–∞–≤—à–∏–µ—Å—è 2 –¥–∏—Å–∫–∞ –Ω–∞ –ø–æ—Å–ª–µ–¥–Ω–∏–π —Å—Ç–µ—Ä–∂–µ–Ω—å. –ü—Ä–∏ —ç—Ç–æ–º —É –Ω–∞—Å –∑–∞–¥–µ–π—Å—Ç–≤–æ–≤–∞–Ω–æ –º–æ–∂–µ—Ç –±—ã—Ç—å —Ç–æ–ª—å–∫–æ **3 —Å—Ç–µ—Ä–∂–Ω—è** (—Ñ–æ—Ä–º—É–ª–∞ **`2^n ‚àí 1`**).
3. –ü–µ—Ä–µ–Ω–æ—Å–∏–º `k`  –¥–∏—Å–∫–æ–≤ –Ω–∞ –ø–æ—Å–ª–µ–¥–Ω–∏–π —Å—Ç–µ—Ä–∂–µ–Ω—å –∏—Å–ø–æ–ª—å–∑—É—è –≤—Å–µ **4 —Å—Ç–µ—Ä–∂–Ω—è** (–ø–æ–≤—Ç–æ—Ä—è–µ–º –ø.1).

–ü–µ—Ä–µ–Ω–æ—Å–∏–º –¥–∏—Å–∫–æ–≤: `2*`(`k` –¥–∏—Å–∫–æ–≤(4 —Å—Ç–µ—Ä–∂–Ω—è)) **+** `n - k` –¥–∏—Å–∫–æ–≤ (3 —Å—Ç–µ—Ä–∂–Ω—è).
–§–æ—Ä–º—É–ª–∞: `2*(2^k-1 + 1) + 2^n-k - 1`.
–î–ª—è `T(n) = T(4)` -> `k = n - 2`.
–ü–æ–ª—É—á–∞–µ–º: `2*(2^(n - 2)-1 + 1) + 2^n-(n-2) - 1` = `2*(2^n-3 + 1) + 2^2 - 1` =
= `((2^n-3+1 + 2*1) + 4 - 1` = `2^n-2 + 5`.

`T(4) = 2^4-2 + 5 = 9`.

```java
    int numLength = numerics.length;  
      
    if (numLength == 1) {  
        return numerics[0];  
    }  
  
    int [] firstPart = merge(Arrays.copyOfRange(numerics, 0, numLength/2));  
    int [] secondPart =
			    merge(Arrays.copyOfRange(numerics, numLength/2, numLength));  
  
    int [] concat = new int [firstPart.length + secondPart.length];  
    int countFirst = 0;  
    int countSecond = 0;  
  
    for (int i = 0; i < concat.length; i++) {  
        boolean firstEnd = countFirst == firstPart.length;  
        boolean secondEnd = countSecond == secondPart.length;  
  
        if (firstEnd && !secondEnd) {  
            concat[i] = secondPart[countSecond++];  
            continue;  
        } else if (secondEnd && !firstEnd) {  
            concat[i] = firstPart[countFirst++];  
            continue;  
        }  
  
        if (firstPart[countFirst] < secondPart[countSecond]) {  
            concat[i] = firstPart[countFirst++];  
        } else {  
            concat[i] = secondPart[countSecond++];  
        }  
    }  
  
    return concat;  
}
```

```java
public static int[] sort(int[] arr) {  
    if (arr.length < 2)  
        return arr;  
  
    int[] first = sort(Arrays.copyOfRange(arr, 0 , arr.length / 2));  
    int[] second = sort(Arrays.copyOfRange(arr,arr.length / 2, arr.length));  
  
    return merge(first, second);  
}  
  
private static int[] merge(int[] f, int[] s) {  
    int[] concat = new int[f.length + s.length];  
  
    int fCount = 0;  
    int sCount = 0;  
  
    for (int i = 0; i < concat.length; i++) {  
        boolean fEnd = fCount == f.length;  
        boolean sEnd = sCount == s.length;  
  
        if (!fEnd && sEnd) {  
            concat[i] = f[fCount++];  
        } else if (fEnd && !sEnd) {  
            concat[i] = s[sCount++];  
        } else if (f[fCount] < s[sCount]) {  
            concat[i] = f[fCount++];  
        } else {  
            concat[i] = s[sCount++];  
        }  
    }  
  
    return concat;  
}
```

```java
private static void sort(int[] arr, int begin, int end) {  
    // end - begin < 2 - –Ω–µ –ø–æ–¥—Ö–æ–¥–∏—Ç, —Ç.–∫. —ç—Ç–æ –∏–Ω–¥–µ–∫—Å—ã,
    // –∞ —Ä–∞–∑–Ω–æ—Å—Ç—å —Å–æ—Å–µ–¥–Ω–∏—Ö –∏–Ω–¥–µ–∫—Å–æ–≤ –¥–∞—Å—Ç 1.  
    
    // –∞ —Ç–∞–º –Ω–∞—Ö–æ–¥–∏—Ç—Å—è 2 —ç–ª-—Ç–∞ –ø—Ä–∏ —ç—Ç–æ–º.

    int mid = begin + (end - begin) / 2;  
    int m = arr[mid];  
  
    int i = begin;  
    int j = end;  
  
    while (i <= j) {  
        while(m > arr[i]) i++;  
        while(m < arr[j]) j--;  
  
        if (i <= j) {  
            int temp = arr[i];  
            arr[i] = arr[j];  
            arr[j] = temp;  
  
            i++;  
            j--;  
        }  
    }  
  
    if (begin < j) sort(arr, begin, j);  
    if (end > i) sort(arr, i, end);  
}
```
